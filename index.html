How It Works

📂

Dataset-Powered RAG

Loads 3,001 real debate entries from the AI jobs dataset. Each argument is tagged with topic, label (pro/con/neutral), source type, year, sentiment, and argument strength scores.

CSV ParsingIn-BrowserNo Backend
🔍

TF-IDF Retrieval

A lightweight TF-IDF similarity engine scores every document against your query. Top relevant documents are retrieved and split by label to form the debate context.

TF-IDFCosine SimilarityTop-K Retrieval
⚖️

Argument Scoring

Arguments are ranked by argument_strength × relevance score. Each side shows the top 5 strongest arguments with source metadata and strength indicators.

Strength ScoreSentimentYear-weighted

Runs entirely in your browser. No Python server, no OpenAI API key, no installations. Just open the HTML file and start debating. Chart.js for analytics visualizations.
